# stress-weighted-sparse-saa

Stress-weighted sparse strategic asset allocation with soft-Frobenius covariance estimation, robust log-growth constraints, and greedy core-satellite portfolio construction.

This repository accompanies the paper:

> **Stress-Weighted Sparse Strategic Asset Allocation: Soft-Frobenius Covariance Estimation, Robust Growth Constraints, and Greedy Core-Satellite Construction**  
> Saverio Lauriola

## Research question

Can strategic asset allocation be improved by conditioning risk estimates on states in which the dependence structure across asset classes departs from its long-run configuration?

The project treats the problem as **risk estimation under diversification failure**, not return forecasting. The framework estimates a stress-conditioned covariance matrix using a continuous Frobenius-distance regime signal, then constructs sparse core-satellite portfolios under robust log-growth constraints.

## Methodology

The framework combines:

1. **Frobenius dependence-instability signal**: distance between current and long-run correlation matrices.
2. **Soft stress weighting**: exponential reweighting of historical observations instead of binary stress/non-stress classification.
3. **Shrinkage-stabilized stress covariance**: stress-conditioned covariance with estimation-error control.
4. **Robust log-growth constraint**: lower-bound constraint based on stationary-bootstrap growth estimates.
5. **Greedy core-satellite allocation**: sparse, interpretable portfolio construction around an equity core.
6. **Validation and robustness**: walk-forward tests, ablation, transaction-cost robustness, subperiod analysis, and block-bootstrap inference.

## Current headline evidence

The main implementable evidence comes from validation-selected rolling and expanding walk-forward tests, net of data-driven transaction-cost estimates. The framework improves path-dependent drawdown metrics, especially Ulcer Index and Martin Ratio, across walk-forward protocols and an eight-model transaction-cost panel.

In the expanding walk-forward protocol, Greedy Soft-Frob improves Martin Ratio from **1.52 to 2.44** versus Markowitz Base-Cov and reduces maximum drawdown from **-22.3% to -17.8%**.

The paper reports raw return improvements as positive but statistically weaker; the contribution is a **risk-estimation and compound-growth preservation result**, not an alpha claim.

## Running the notebook

Open:

```text
notebooks/stress-weighted-sparse-saa.ipynb
```

Run cells top-to-bottom. The notebook:

1. downloads ETF data via `yfinance`;
2. computes regime indicators and stress-weighted covariance estimates;
3. solves greedy and continuous portfolio-construction variants;
4. runs validation-selected walk-forward tests;
5. estimates transaction-cost robustness;
6. produces figures, tables, subperiod analysis, and inference diagnostics.

Some sections are computationally expensive, especially walk-forward validation and bootstrap inference. Execution flags are collected in the global-parameters cell.

## Methodological caveats

- ETF proxies are used as transparent investable asset-class representatives.
- Results depend on the data vendor, universe, liquidity assumptions, validation design, and transaction-cost model.
- The greedy algorithm is an interpretable sparsification layer, not a formal global optimizer.
- The Frobenius signal measures dependence-structure instability; it is not a crisis predictor.
- Transaction-cost estimates are based on daily OHLC data and should be interpreted as approximations.
